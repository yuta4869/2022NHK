File Operations

Click on an icon for more details

Create a new file

This option opens the Material Setup form, which is used to create a new blank job to the width (X), height (Y), and thickness (Z) required. The relative origins for X0, Y0 and Z0 are also specified at this point, and the measurement units can be set in either inches or metric.

 

 

 

Width

The material length along the X axis
 

Height

The material length along the Y axis

  Thickness The material thickness Z axis
  Z Zero

Indicates whether the tip of the tool is set off the surface of the material (as shown in the diagram) or off the bed / table of the machine.

  XY Origin

The X0, Y0 origin for the job for setting the CNC machine and can be set at any corner or the middle of the job.

  Use origin offset

Moves the material origin(bottom left or middle) relative to absolute X0, Y0.

In most cases a new job represents the size of the material the job will be machined into.

   

Clicking OK creates a new empty job, which is drawn as a gray rectangle in the 2D Design window.

   

Dotted horizontal and vertical purple lines are drawn in the 2D design window to show where the X0 and Y0 point is positioned.

Open file

This option opens the File Open dialog window, allowing existing Cut2D files (CRV extension) and importable DXF (2D), EPS and AI files to be selected and opened.

 

 

  CRV

Files previously created and saved in Cut2D will be opened and displayed in the 2D Design window. All calculated toolpaths are stored / opened from the CRV file.

  DXF

Files from other CAD or graphics software packages such as AutoCAD will be opened in the original size and position. The Material Setup Form is automatically opened showing the maximum X & Y dimensions of the opened design. The actual size of the material can then be specified along with the required thickness and appropriate X0, Y0 and Z0 origins.

  EPS

Files from typical design software such as Corel Draw can be opened. The Material Setup form will automatically be opened so the required material size can be specified. By default the EPS file will be placed with the lower left corner of the design at X0, Y0.
  AI

Files from typical design software such as Corel Draw can be opened. The Material Setup form will automatically be opened so the required material size can be specified. The AI file will be placed with the lower left corner of the design at X0, Y0

 

PDF

Software such as the Adobe product range can be used to convert files from other design and word processing software into the industry standard PDF file format. The text and vector content of PDF files is extracted when imported into Cut2D.

The Material Setup form will automatically be opened so the required material size can be specified. The AI file will be placed with the lower left corner of the design at X0, Y0

All data from pages in multiple page PDF documents will be imported into the file.

     

 

Width (X)

By default the actual width of the data in the DXF file is displayed. This dimension can be edited to be the required material size and the vectors scaled accordingly.

 

Height (Y)

By default the actual height of the data in the DXF file is displayed. This dimension can be edited to be the required material size and the vectors scaled accordingly.

 

Thickness

The material thickness Z axis

 

Z Zero

Indicates whether the tip of the tool is set off the surface of the material (as shown in the diagram) or off the bed / table of the machine.

 

XY Origin

The X0, Y0 origin for the job for setting the CNC machine and can be set at any corner or the middle of the job.

 

Use origin offset

Moves the material origin(bottom left or middle) relative to absolute X0, Y0.

 

Center vectors in material

Important!

Checking this option automatically moves the vectors in the DXF file into the middle of the specified material block.

Do not select this option if the job needs to stay in its original drawn position

 

Information

The material size for a job can be changed at any time by selecting Edit > Material size from the main menu along the top of the interface.

This option opens the File Save As dialog window and allows the job to be saved as a .CRV Cut2D file. Navigate to the required folder, enter a suitable name for the job and click the Save button

Notes

Before exporting EPS or AI files from software programs such as Corel Draw, convert the text objects to Curves. This is typically done by selecting,

Arrange > Convert to Curves

 
    When importing EPS, AI and PDF files, any image data in the file will not be imported. Only vector entities will be extracted from the file.
    DXF files from CAD systems such as AutoCAD will often appear as individual lines and arcs after being imported. Use the Join command to join the lines and arcs back together so they can be machined efficiently.

 

3D Files

Files from Cut3D, PhotoVCarve and Vector Art 3D Machinist that include 3D toolpaths can be imported into Cut2D.

The 3D file must first be scaled to the required size, toolpaths calculated and then the complete file saved ready for importing into Cut2D. These files can only be moved and positioned inside Cut2D and cannot be scaled.

From the File Operations area on the Drawing Tab, select the Import icon and the PhotoVCarve PVC and VA3D-Machinist V3M files can be selected. Alternatively, from the Main Menu click,

File > Import

A Grayscale thumbnail of 3D job is drawn in the 2D window with the X0 Y0 origin at the position it was set in Cut3D, PhotoVCarve or VA3D-Machinist. The associated toolpath(s) are also drawn in the 3D window and the names appear in the Toolpath list.
 
Positioning

To move the 3D design toolpaths,

  1. Open the 2D Window
  2. Click the Left mouse Twice on grayscale image (turns light Blue to indicate it's selected)
  3. Drag to the required position
  4. Or use the Move or Alignment tools for accurate positioning.

The toolpath(s) are automatically moved in the 3D window to the same XY position as the image.

   

Origin for the 3D data at X0 Y0

Grayscale image moved to middle of job

   

Toolpaths for the example above have been calculated with the X0 Y0 in the middle of the 3D design. When imported into Cut2D the data is automatically positioned using the same coordinates, which places three quarters of the design off the job. In the second image the grayscale image has been moved to the middle of the job.

The 2D mirror and rotate drawing tools to can also be used to edit the 3D data set.

3D toolpaths can also be Copied using the Toolpath Operations making it very easy to use multiple elements from a single design on a job. The thumbnail preview is also copied for each toolpath, making it very easy to position additional copies of a 3D toolpath.

For example, a single design can be copied and mirrored to create Left and Right versions of a 3D design or to place multiple copies of a decorative design in the corners of a cabinet door panel as shown below.

   

The 4 corner roses are all copied from a single file

   

Toolpaths for the 3D elements can be previewed along with the conventional Profile, Pocketing and Drilling toolpaths, and everything saved ready for machining.

A good example of where this functionality might be used in conjunction with PhotoVCarve is for making personalized picture frames that include the PhotoVCarve grooves plus descriptive engraved text and a decorative Profiled or Beveled border. As shown below,

   

2D window showing thumbnail previews, text and decorative border

   

3D window previewing the completed job

 
Options

Imported toolpaths can also be edited to position them inside the material or to change the cutting parameters - speeds and feedrates can be changed.

Click the Edit toolpath icon or Double click on the toolpath name to open the edit form.

   
   

For example, after machining a 0.5" deep pocket a PhotoVCarve design can then be edited to have a Start Depth = 0.5" and this will carve the photograph onto the base of the pocket surface.

Important When using a VA3D-M file that includes multiple toolpaths, you must remember to edit the Start Depth for all of the imported 3D toolpaths.

Recent files

This area on the Startup tasks area of the interface displays a list of the previous 5 files that have been opened, allowing them to be instantly re-opened by simply clicking the left mouse button on the file name.

Cut

The Cut tool removes the selected vectors from a design in a very similar way as pressing the Delete key. But the selection details are also copied to the clipboard and can be Pasted into a new file if required.

Copy

The Copy tool copies selected vectors to the clipboard, leaving the original in place and allows duplicates to be made and re-used in the design by pressing the Paste icon.

Paste

This Paste option places the contents of the clipboard (created by Cut or Copy) into the design, allowing elements to be re-used in different areas of a design.

Edit Undo / Redo

Edit undo / redo can be used to step backwards or forwards through the last 50 design changes.

 

Notes

Clicking twice, holding Ctrl and Dragging with the Left mouse button creates a copy of the selection when the mouse button is released.

Clicking the Right mouse button displays the Quick Edit menu that allows easy access to the Cut, Copy, Paste, Edit Undo and Edit Redo commands.

Node Edit

The Node Editing tool can be selected from the Editing window or by pressing the Keyboard shortcut N to toggle between Selection and Node Editing modes.

 

 

Interactive node editing allows designs to very easily be modified. Node editing is particularly useful for 'fixing' geometry that includes problems such as loops or crossings that may cause machining issues.

   

Select the Node Editing icon or press N to start Node editing.

Pressing N will toggle between Selection and Node Editing modes.

Pressing Esc cancels Node Editing and returns to Selection mode.

   

The selected vector is displayed showing the individual nodes for lines, beziers and arcs. When in Node Edit mode the cursor can be used to select single or multiple nodes.

Selected nodes are drawn as Red boxes

  Types of Nodes

A Black node indicates the point represents a discontinuity / sharp corner

A Blue node indicates the line passing through the point will be smooth

A White node indicates the mid point on an arc or bezier handle

  Multiple node selection

Multiple nodes can be selected by,

Holding the Shift key and clicking the Left mouse button on required nodes

Dragging a box to include the required nodes

    Nodes can be,

Moved by clicking and dragging with the Left mouse button

Moved / Nudged using the Arrow keys (+ Ctrl smaller and Shift larger increments)

Deleted by pressing the D key

Multiple nodes can be aligned horizontally, vertically or simply in a straight line between the start and end nodes using keyboard shortcuts.

When in Node Edit mode select multiple nodes and press,

            X to Align vertically

            Y to Align horizontally

            L to Delete intermediate nodes to leave a single straight line

  Editing

Placing the cursor over a node and Right clicking displays the menu options available for the node.

Delete Point removes the selected node

Smooth Point toggles between Smooth (Blue node) / Unsmooth (Black node)

Cut Vector opens the vector at the point leaving 2 nodes on top of each other

Make Start Point for machining purposes

The cursor changes to display '~' to indicate that it's positioned over a curve into which a new point / node can be inserted.

Join

Open vectors are automatically identified and closed or joined to other vectors where the end points lie within the user definable tolerance.
 

 

   

1. Select all the vectors to check (Ctrl +A will select all vectors in the design)

The form shows the number of Closed and Open vectors in the selection.

2. Enter the required tolerance.

The form updates to show the number of vectors that will be closed based on the tolerance.

3. Click the Join button

The end point on each vector is moved to the start point, closing the vector.

Join Vectors

An open vector can be joined to another open vector using the options below.
 

Join with a Line finds the closest end points on 2 selected, open vectors and joins with a straight line.
 

Join Moving End Points finds the closest end points on 2 selected, open vectors, calculates the mid point between them and moves the end points to this position.

 

Close Vectors

Profile Toolpaths

Profile Machining options provide the flexibility for cutting shapes out with optional Tabs / bridges plus an Allowance over / undercut to ensure perfect edge quality.

The 3D Bevel machining strategy chamfers the edge of shapes and includes 3D sharpening moves into corner regions, that makes the finished parts look perfect.

Profile toolpaths can be outside, inside or on the selected closed vectors, automatically compensating for the tool diameter and angle. These machining strategies can use any of the cutter types supported and the toolpaths will automatically offset to compensate for the geometry of the selected tool.

When working with open vectors the profile toolpaths can be to the Left, to the Right or On the selected vectors.
   
  Cutting Depths

Start Depth (D) specifies the start depth at which the profile toolpath is calculated.
   

When cutting directly into the surface of a job the Start Depth will often be 0.

If machining into the bottom of an existing pocket or stepped region, the depth of the pocket / step must be entered.

For example, to machine a groove into the bottom of a 0.5” deep pocket the Start Depth = 0.5”
   

Cut Depth (C) is the depth of the pocket or profile toolpath relative to the Start Depth.

Drilling

Drilling allows the centers of selected closed vectors to be drilled to a specified depth. The Tool database includes an option to specify the Drill diameter and cutting parameters.

 

 

  Peck Drilling

When the option for Peck Drilling is selected, the drill will cut to the Pass Depth set in the Tool database for the selected Drill. Then retract to the Retract Gap above the surface and then continue drilling to an additional Pass Depth. This cycle is repeated until the hole is drilled to the required depth. The retract moves are used to remove any build-up of material from the hole to help with overheating and breakage.

If the Pass Depth is greater than the required Cut Depth, the hole will be drilled in a single cycle. However, if a Cut Depth of 1" with a Pass Depth of 0.25" is used the hole will be drilled in 4 cycles.

Cutting Parameters

Pass Depth is the maximum depth of cut the tool can cut. The Pass Depth controls the number of z level passes that are calculated for a toolpath. For example, creating a pocket 1" (24 mm) deep using a tool that has a Pass Depth of 0.25" (6 mm) will result in the toolpath making 4 passes.

Stepover is the distance the cutter moves over when doing area clearance cutting. For example,when raster machining the cutter will machine along the X axis, stepover in the Y direction and return paralle to the first line of cut. The greater the stepover the faster the job will be machined, but this must be balanced with the material being cut and the tooling being used, to ensure that the tool does not break.

When stepovers greater than 50% of the cutter / tip diamter are used the software automatically adds 'Tail' moves the corner regions on toolpaths to ensure material is not left on the job.

 
 

When using V-Bit Tools the Stepover fields automatically change to use the following options.

Final Pass Stepover is the distance the cutter moves over when finish machining and is usually set to be a relatively small distance to produce a smooth surface finish on the job.

Clearance Pass Stepover is only used when a V-Bit tool is being used to rough machine at multiple Z levels down to a specified flat depth. This stepover can be much larger than the Final Pass Stepover because the tool is only rough machining material away. Increasing the Clearance Pass Stepover will reduce the machining time, but you must be careful to ensure it is not too great for the material being cut.

 
  Feeds and Speeds

Spindle Speed is specified in revolutions per minute

Feedrate is the surface cutting rate at which the cutter is moved into the material. The units can be specified in distance per second or minute.

Plunge Rate is the cutting rate at which the cutter is moved vertically into the material. The units can be specified in distance per second or minute.

The Feedrate and Plunge rate will vary depending upon the material being machined and the tooling being used.

 

Tool Number

This is the number of the tool needed to machine the job.
